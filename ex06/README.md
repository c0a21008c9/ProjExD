# 第6回
## エアホッケー（ex06/newgame.py)
### ゲーム概要
- ゲームを開始すると1600×900のフィールドが現れる、そこでエアホッケーをするゲーム
- 敵の操作するキーパー側の面にボールが触れると＋１点
- どちらかが５点を取ったらゲームセット
### 操作方法
- プレイヤー１はW、Sでプレイヤー２はUP,DOWNでキーパーを操作する
### 追加機能
- ゲームの点数を表示
- BGMを追加（著作権フリー）
- ゲーム終了時の効果音を追加(著作権フリー)
- ゲーム中の効果音を追加(著作権フリー)
- 障害物のランダムな位置に生成
